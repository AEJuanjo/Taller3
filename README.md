## ⦗📁⦘ Repositorio:

>En este repositorio están los archivos que explican practicamente el uso de los ciclos y mas herramientas de codigo, las series de Fibonacci, N-avos numeros primos, raices de ecuaciones cuadraticas y promedios a numeros.

## ⦗🧑🏼‍💻⦘ Programa:

>El programa de java cuenta el uso de ciclos, ya sean while, for, if, else if y demas. Tambien, cuenta con un menu que nos permite realizar multiples operaciones e interactuar de manera directa con operaciones de calculo, programadas en el mismo.

>El programa esta en github.


## ⦗🌐⦘ Programa de las ecuaciones:

>Se evidencia el codigo con un if el cual permite escoger entre 3 opciones, opcion 1 para la ecuacion 1, la opcion 2 para escoger la ecuacion 2, y la opcion 3 para salir del sistema, adicionalmente , se incluye la opcion de que si el usuario ingresa una ocpion no valida, el sistema devuelva un mensaje indicando que la entrada que ingreso el usuario no es valida y que debe volver a intentarlo

## ⦗🔢⦘ Programa de la serie Fibonacci:

Se evidencia el código utilizando un ciclo for el cual permite generar los n primeros números de la serie de Fibonacci a partir de un número ingresado por el usuario. El programa muestra cada número de la serie y calcula de manera progresiva la suma acumulada de los valores generados, hasta finalmente mostrar la suma total de los números de Fibonacci calculados.

## ⦗🧮⦘ Programa del N-esimo número primo:

Se evidencia el código mediante ciclos while y for que permiten encontrar el N-ésimo número primo según la posición ingresada por el usuario. El programa evalúa números consecutivos y determina si cada uno es primo verificando la cantidad de divisores que posee. Cuando se encuentra el número primo correspondiente a la posición indicada, el sistema lo muestra como resultado.
